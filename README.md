Here is the code accompanying the manuscirpt "The critical dynamics of hippocampal seizures"

The code for the in silico simulation using the Epileptor are provided as 
Jupyter notebook to allow to reproduce easily the analysis and plots 
presented in the paper.

For the in vivo analysis, all the main functions to analyse and
plot the data are provided as python files.
